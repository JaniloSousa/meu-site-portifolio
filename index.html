Site com Menu Hamburguer

Visando melhorar a visualização dos meus sites em dispositivos móveis aprendi a criar um menu hambúrguer responsivo ultilizando a filosofia do mobile first. Esse efeito faz o menu principal do site, inicialmente disposto horizontalmente, "se contrair" e um único ícone que se assemelha bastante a um hambúrguer quando as dimensões da tela atingem um determinado tamanho. Somente clicando nesse novo ícone o menu reaparece agora disposto verticalmente em formato de colunas com cada opção uma embaixo da outra. Excelente projeto para ter seu primeiro contato com media queries e javascript pois o projeto usa dessas duas tecnologias mas de maneira bem simples.

Site Com Efeito Smooth Scroll

Um efeito bem legal e elegante para seus sites, e particularmente bem simples de se fazer. Ao clicar em uma das opções o usuário será "levado" para a região do site correspondente com um efeito 'suave' de scroll evitando assim a rolagem excessiva e desnecessária do scroll do mouse.

Calculadora em JavaScript

Não podia faltar esse clássico projeto de iniciantes. Embora muito clichê, não deixar de ser um excelente projeto para treinarmos o DOM da linguagem javascritp. Nesse projeto aprendemos a lidar com o disparo de eventos dessa linguagem, já que ela é uma linguagem orientada a eventos, e a usar tabelas para construir a interface gráfica da calculadora. Catalogo de Filmes (consumindo uma API com JavaScript)

Além de saber lidar com eventos disparados via DOM da linguagem javascript, um desenvolvedor também precisar saber lidar com as várias API's que estão aí disponíveis no mercado, que contém milhares de informações interessantes só esperando para serem consumidas. Nesse projeto, consumi a api da IMDB que retorna um array com vários títulos de filmes e algumas outras informações sobre eles. CRUD em Python + MySql (Sistema Escolar Cursos e Alunos)

Programa feito em Python para rodar em Desktops. Usei a biblioteca nativa do python, Tkinter, para construir a interface gráfica. O programa visa simular um sistema escolar de cadastro de alunos e de cursos que poderão ser assistidos por esses alunos, com as quatro opções do CRUD. Utilizei a biblioteca SQL Alchemy como ORM para não ter que ficar inserindo a sintaxe do MySql dentro do código do programa. Utilizei majoritariamente o paradigma de Programação Orientada a Objetos para escrever o programa. Fiz ele visando treinar e provar meus conhecimentos em GUI, Banco de Dados e relacionamento entre tabelas.
