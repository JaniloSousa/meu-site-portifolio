Site Com Efeito Parallax

Um efeito muito legal e bonito, que vários sites atualmente usam. O efeito parallax consistem no usuário ir descendo a página enquanto o fundo do site permanece fixo e vai sumindo conforme o scroll, além de trazer um ar elegante para seus sites é um ótimo projeto iniciante para treinar HTML e CSS, principalmente CSS no que se refere as propriedades de 'background'.

Site com Menu Hamburguer

Visando melhorar a visualização dos meus sites em dispositivos móveis aprendi a criar um menu hambúrguer ultilizando a filosofia do mobile first. Esse efeito faz o menu principal do site, disposto horizontalmente, "se contrair" e um único ícone que se assemelha bastante a um hambúrguer. Ao clicar nele só então o menu reaparece agora disposto verticalmente em formato de colunas com cada opção uma abaixo da outra. Excelente projeto para ter seu primeiro contato com media queries e javascript pois usam bem pouco dessas tecnologias.

Visualizar

Site Com Efeito Smooth Scroll

Um efeito bem legal e elegante para seus sites, e particularmente bem simples de se fazer. Consiste em linkar as opções apresentadas no cabeçalho do seu site a certas sessões da página atualmente aberta. Ao clicar em uma das opções o usuário será "levado" para a região do site correspondente com um efeito 'suave' de scroll evitando assim a rolagem excessiva e desnecessária do scroll do mouse.

Visualizar

Calculadora em JavaScript

Não podia faltar esse clássico projeto de iniciantes. Embora muito clichê, não deixar de ser um excelente projeto para treinarmos o DOM da linguagem javascritp. Nesse projeto aprendemos a lidar com o disparo de eventos dessa linguagem, já que ela é uma linguagem orientada a eventos, e a usar tabelas para construir a interface gráfica da calculadora. Também aprendemos a utilizar a função 'eval()' para o cálculo de expressões numéricas que estão inicialmente em formato de string.

Catalogo de Filmes (consumindo uma API com JavaScript)

Além de saber lidar com eventos disparados via DOM da linguagem javascript, um desenvolvedor também precisar saber lidar com as várias API's que estão aí disponíveis no mercado, que contém milhares de informações interessantes só esperando para serem consumidas. Nesse projeto consumi a api da IMDB que retorna um array com vários títulos de filmes e algumas outras informações sobre eles. Excelente projeto para treinar o método 'fetch()' e também para mais tarde ajudar você a desenvolver o projeto tão cobiçado do 'Clone da Netflix'.
